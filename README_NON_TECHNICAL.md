# Non-Technical Guide: MVP Chat App

## What is this?
This is a modern, high-fidelity chat application prototype designed to look exactly like the provided Figma concepts. It features a clean, conversion-focused interface, secure Google Login, and a demonstration of AI chat capabilities.

## Key Features

### 1. Beautiful Design (Figma Match)
- **Mint/Teal Theme**: A custom color palette designed for a calming, premium feel.
- **3-Column Layout**: Easy navigation with Sidebar, Chat List, and Chat Window.
- **Responsive Animations**: Smooth hover effects and transitions.

### 2. "Chat with AI" Demo
- **AI Assistant**: You can view an "AI Assistant" in your chat list.
- **Simulated Intelligence**: Send a message to the AI, and it will reply (currently simulated for demonstration without costs).

### 3. Secure Login
- **Google Authentication**: Log in quickly and securely with your Google account.
- **Privacy First**: No passwords stored locally.

## How to Run
1. Open the project folder.
2. Run `npm run dev` in your terminal.
3. Open `http://localhost:3000` in your browser.

## Tech Stack (For Developers)
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS v4
- **Database**: Prisma (Ready for Postgres)
- **Auth**: Google OAuth
